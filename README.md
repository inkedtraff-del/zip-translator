# INKED TRANSLATOR PRO 3000 (Vercel)

Это статический сайт, который переводит **только текст внутри HTML** (теги/структура сохраняются) через серверную функцию **`/api/translate`**.

## Деплой на Vercel (пошагово)
1) Создай репозиторий на GitHub и загрузи в него файлы из этого архива.
2) Зайди на Vercel → **New Project** → выбери репозиторий → нажми **Deploy**.
3) В Vercel открой проект → **Settings → Environment Variables** и добавь:
   - **OPENAI_API_KEY** = `sk-...`
   - (опционально) **OPENAI_MODEL** = `gpt-4.1-mini`
4) Нажми **Save**. Vercel сам пересоберёт проект.

Готово: открой сайт, загрузи HTML, выбери язык/страну/валюту и скачай `index.html`.
